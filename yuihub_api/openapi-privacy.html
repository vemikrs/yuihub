YuiHub API プライバシーポリシー

最終更新日: 2025年9月26日

1. 概要

YuiHub API（以下「本サービス」）は、AI会話メモリ横断基盤として、会話データの保存・検索・文脈生成機能を提供します。本プライバシーポリシーは、本サービスにおけるデータの取り扱いについて説明します。

2. 収集するデータ

本サービスでは以下のデータを収集・処理します：

  • AI会話の内容（テキストデータ）
  • メタデータ（タイムスタンプ、タグ、スレッドID等）
  • API利用ログ（アクセス時刻、IPアドレス、リクエスト内容）
  • 認証情報（APIトークン）

3. データの利用目的

収集したデータは以下の目的で利用します：

  • 会話メモリの保存・検索サービスの提供
  • 文脈情報の生成・共有
  • サービスの改善・最適化
  • 不正利用の防止・セキュリティの確保

4. データ保護措置

本サービスはPoC（概念実証）環境として、以下のShelterモード設定で動作します：

  • 外部IO制限: EXTERNAL_IO=blocked
  • ローカルストレージのみ使用
  • 最小限の認証機能
  • 暗号化通信（HTTPS）の利用

5. データの保存期間

収集されたデータは、サービス提供に必要な期間中保存されます。削除の要請がある場合は、技術的に可能な範囲で対応いたします。

6. 第三者提供

法令に基づく場合を除き、収集したデータを第三者に提供することはありません。PoCモードでは外部システムとの連携は制限されています。

7. ユーザーの権利

ユーザーは以下の権利を有します：

  • 保存されたデータの確認・修正の要求
  • データの削除要求
  • データ処理の停止要求

8. 本ポリシーの変更

本プライバシーポリシーは、法令の変更やサービス改善に伴い変更される場合があります。重要な変更がある場合は、適切な方法で通知いたします。
